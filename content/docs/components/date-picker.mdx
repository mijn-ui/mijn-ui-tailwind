### DatePicker Structure

The DatePicker component is composed of two main components:

1. **[Input](/docs/components/input)**: This component handles the user input for selecting a date.
2. **[Calendar](/docs/components/calendar)**: This component displays a calendar interface for date selection.

For more detailed information, please refer to their respective documentation.
